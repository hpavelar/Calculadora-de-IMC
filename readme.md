# Calculadora de IMC
![tela da aplicação](./tela.JPG)

## Objetivo
Calcular o IMC do usuário de acordo com a altura e o peso fornecidos, e verificar em que intervalo está o resultado.

## Tecnologias utilizadas
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>
